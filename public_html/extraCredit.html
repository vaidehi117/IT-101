Web Page Do's and Dont's

Do's of Creating a Webpage

1. Information is found easily(easy navigation)

2. Accessible to all users

3. Web site is easily used for it's purpose

4. make sure images are not broken

5. Website loads fast

Dont's of Creating Webpage

1. Non-scannable text

2. Bad Navigation

3. Fixed font size

4. Not changing the color of visited links

5. Page titles with low search engine visibility
